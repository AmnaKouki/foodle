À propos

Ce projet a été créé par Amna Kouki pour aider les consommateurs à trouver le meilleur prix possible lors de l'achat de produits en ligne. J'ai utilisé les technologies Python, Flask, HTML, CSS, JavaScript et Tailwind pour créer une application Web facile à utiliser qui permet aux utilisateurs de comparer les prix de produits sur différents sites d'achat en ligne. Ce site a été conçu pour être responsive et "Mobile-Friendly" afin de garantir une expérience de navigation optimale pour tous les utilisateurs, quel que soit leur appareil.

L'objectif de ce projet est de fournir une expérience de shopping en ligne transparente et efficace en donnant aux utilisateurs un accès facile aux prix des produits sur différents sites. J'espère que notre projet sera utile pour les consommateurs qui cherchent à économiser de l'argent lors de leurs achats en ligne.
